# Stock & Crypto Data Warehouse

## Overview

This project implements a Data Warehouse for Stock and Cryptocurrency market data using PostgreSQL. The system collects, transforms, and stores historical market data through ETL/ELT pipelines to enable efficient analytical querying and reporting.

The project focuses on dimensional modeling techniques and evaluates the performance and cost implications of different SQL join strategies for large-scale financial datasets.

## Objectives

- Build a centralized data warehouse for stock and cryptocurrency data.
- Design ETL/ELT pipelines for data extraction, transformation, and loading.
- Implement dimensional modeling for analytical workloads.
- Optimize data retrieval using efficient SQL queries.
- Analyze the performance of different SQL join strategies.

## Technologies Used

- PostgreSQL
- SQL
- ETL / ELT Pipelines
- Dimensional Modeling

## Project Architecture

Raw Data → ETL/ELT Pipeline → PostgreSQL Data Warehouse → Analytical Queries → Insights
